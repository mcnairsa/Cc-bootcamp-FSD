About CSS Grid

CSS Grid Layout is a two-dimensional layout method for the web. It lets you create complex responsive web design layouts more easily and consistently across browsers.

Unlike flexbox which is largely a one-dimensional system, Grid Layout is two-dimensional, meaning it can handle both columns and rows. You can work with Grid Layout in two ways - you can either define a grid using explicit measurements or you can just define areas and let the grid auto-fit around your content.
