Em vs Rem Units Comparison

Key Difference: em units scale relative to their parent element's font size, while rem units scale relative to the root element's font size.

Comparison Table

Aspect Em Units Rem Units
Relative to Parent element's font size Root element's font size
Nesting behavior Compounds/multiplies with nesting Consistent across all levels
Best for Component-based spacing, buttons, icons Typography, layout, consistent sizing
Predictability Less predictable in deep nesting Always predictable
Use cases Padding/margins within components Font sizes, breakpoints, grid gaps

Practical Examples

When to Use Em

Use em units for:

  • Padding and margins within components
  • Button sizing (so they scale with text)
  • Icon sizes relative to text
  • Border-radius that should scale

When to Use Rem

Use rem units for:

  • Font sizes for consistent typography
  • Media query breakpoints
  • Layout spacing (margins between sections)
  • Maximum widths and container sizing
